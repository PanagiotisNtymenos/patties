## Overview

This project is a scaffolded command bus pattern implementation using Kotlin.

## Prerequisites

- JDK 11 or higher
- Gradle 6.0 or higher

## Building

To build the project, run the following command:

```sh
./gradlew ktel:build
```

## Tests

To execute the tests, use the following command:

```sh
./gradlew ktel:test
```
